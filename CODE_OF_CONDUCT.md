# Code of Conduct

Nemo Assistant is an open-source project for building a useful local-first desktop assistant. Participation should stay respectful, constructive, and focused on making the project better.

## Expected Behavior

- Be kind and professional.
- Assume good intent, especially across languages and time zones.
- Keep criticism specific, technical, and actionable.
- Respect maintainers' time by providing clear reproduction steps and context.
- Avoid sharing private data, API keys, or screenshots that expose someone else's information.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing another person's private information.
- Repeated low-effort or off-topic disruption.
- Pressuring maintainers to prioritize unpaid work on a specific timeline.

## Enforcement

Maintainers may edit, hide, lock, or remove comments and issues that violate these expectations. Repeated or severe violations may result in being blocked from project spaces.

If you need to report conduct concerns, contact the maintainer privately rather than opening a public issue.

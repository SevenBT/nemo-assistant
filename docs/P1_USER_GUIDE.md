# P1 功能使用指南：预设角色和会话级 System Prompt

## 功能概述

P1 功能为 AI Agent Desktop Assistant 添加了预设角色和会话级 System Prompt 支持，让用户可以：
- 快速创建特定用途的会话（翻译、编程、写作等）
- 为单个会话自定义 AI 行为
- 管理和复用预设角色

## 快速开始

### 1. 使用预设角色创建会话

1. 点击左侧「＋ 新建」按钮
2. 在弹出的对话框中选择预设角色：
   - 🤖 默认助手：通用 AI 助手
   - 🌐 翻译官：中英互译
   - 💻 代码助手：编程辅助
   - ✏️ 写作助手：文案创作
   - 📋 摘要大师：内容总结
3. 点击选择，会话创建完成

### 2. 自定义会话 System Prompt

1. 右键点击会话列表中的会话
2. 选择「会话设置」
3. 选择「自定义 Prompt」单选按钮
4. 在文本框中输入自定义的 System Prompt
5. 点击「保存」

### 3. 切换会话的预设角色

1. 右键点击会话列表中的会话
2. 选择「会话设置」
3. 选择「使用预设角色」单选按钮
4. 从下拉列表中选择预设角色
5. 点击「保存」

### 4. 管理预设角色

1. 点击右上角「⚙️」图标打开设置
2. 切换到「模型」Tab
3. 点击「管理预设角色」按钮
4. 在预设角色管理对话框中：
   - 点击「新建」创建自定义预设
   - 选择预设后编辑名称、图标、提示词
   - 点击「保存」保存修改
   - 点击「删除」删除自定义预设（内置预设不可删除）
   - 点击「导入」从 JSON 文件导入预设
   - 点击「导出」导出自定义预设到 JSON 文件

## System Prompt 优先级

当发送消息给 AI 时，System Prompt 按以下优先级选择：

1. **会话级 system_prompt**（最高优先级）
   - 在「会话设置」中自定义的 Prompt
   
2. **预设角色 preset_id**
   - 创建会话时选择的预设角色
   - 或在「会话设置」中选择的预设角色
   
3. **全局配置 config.system_prompt**
   - 在「设置 → 模型」中配置的全局 System Prompt
   
4. **默认值 DEFAULT_USER_PROMPT**
   - 应用内置的默认提示词
   
5. **追加 BUILTIN_TOOLS_INSTRUCTION**
   - 自动追加工具调用说明（所有会话都会追加）

## 导入/导出预设角色

### 导出格式（JSON）
```json
[
  {
    "id": "python-expert",
    "name": "Python 专家",
    "icon": "🐍",
    "system_prompt": "你是一个 Python 编程专家...",
    "params": {},
    "is_builtin": false
  }
]
```

### 导入步骤
1. 准备 JSON 文件（格式如上）
2. 打开「管理预设角色」对话框
3. 点击「导入」按钮
4. 选择 JSON 文件
5. 导入成功后，预设角色会出现在列表中

### 导出步骤
1. 打开「管理预设角色」对话框
2. 点击「导出」按钮
3. 选择保存位置和文件名
4. 导出的 JSON 文件包含所有自定义预设（不包含内置预设）

## 常见问题

### Q: 如何恢复内置预设角色？
A: 删除 `config/presets.json` 文件，重启应用即可自动重新创建。

### Q: 会话设置会影响其他会话吗？
A: 不会。每个会话的设置是独立的，互不影响。

### Q: 全局 System Prompt 和预设角色有什么区别？
A: 
- 全局 System Prompt：在「设置 → 模型」中配置，影响所有未设置预设的会话
- 预设角色：可以为每个会话单独选择，优先级高于全局配置

### Q: 如何分享我的预设角色？
A: 使用「导出」功能导出为 JSON 文件，分享给其他用户，他们可以使用「导入」功能导入。

### Q: 预设角色的参数（params）有什么用？
A: 目前预设角色的参数功能尚未完全实现，未来可以用于自定义 temperature、max_tokens 等模型参数。

## 技术细节

### 配置文件位置
- 预设角色：`config/presets.json`
- 会话数据：`data/sessions/{session_id}.json`

### 数据结构
```python
# Preset
{
    "id": str,           # 唯一标识
    "name": str,         # 显示名称
    "icon": str,         # 图标（emoji）
    "system_prompt": str,# 提示词
    "params": dict,      # 参数（预留）
    "is_builtin": bool   # 是否内置
}

# Session
{
    "id": str,
    "title": str,
    "preset_id": str,    # 预设角色 ID（新增）
    "system_prompt": str,# 自定义提示词
    "messages": list,
    "created_at": float,
    "updated_at": float
}
```

## 已知问题（来自代码审查）

### MEDIUM 级别
1. **缺少输入验证**：无长度限制，超长输入可能导致 API 错误
   - 建议：添加 2000 字符限制和警告对话框
   - 优先级：P1（第二期优化）

2. **System Prompt 拼接格式**：可能产生多余空行
   - 建议：规范化拼接逻辑
   - 优先级：P2（低影响）

### LOW 级别
1. **缺少字符计数**：用户无法直观了解当前长度
   - 建议：添加实时字符计数标签
   - 优先级：P2（用户体验优化）

2. **恢复默认无确认**：可能误操作
   - 建议：添加确认对话框
   - 优先级：P2（用户体验优化）

## 功能特性

### 1. 直接编辑内置预设
- 所有预设（内置 + 自定义）都可以编辑
- 编辑器区域对所有预设开放
- 保存按钮对所有预设可用

### 2. 修改状态标记
- 被修改的内置预设在列表中显示 ⚠️ 图标
- 例如：`⚠️ 🤖 默认助手`

### 3. 恢复默认功能
- 被修改的内置预设显示"恢复默认"按钮
- 按钮样式：黄色背景（#FEF3C7）
- 点击后弹出确认对话框
- 恢复后 ⚠️ 图标消失，按钮隐藏

### 4. 删除保护
- 内置预设仍然不可删除
- 选中内置预设时"删除"按钮禁用
- 错误提示："内置预设角色不可删除，但可以编辑或恢复默认"

### 5. 原始预设保存
- 系统自动保存所有内置预设的原始版本
- 使用深拷贝确保数据独立
- 恢复时从原始版本复制数据

## 技术亮点

1. **深拷贝保护**：使用深拷贝保存原始预设，防止引用污染
2. **状态同步**：保存后自动重新选中，更新 UI 状态
3. **条件显示**：⚠️ 图标和"恢复默认"按钮根据修改状态动态显示
4. **错误处理**：完善的异常处理和用户提示
5. **数据安全**：删除保护 + 恢复功能双重保障

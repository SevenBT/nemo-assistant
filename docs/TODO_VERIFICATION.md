## 已知限制

1. **重复任务**：当前只保存配置，不自动创建新任务
2. **删除功能**：待办暂不支持删除（可以标记为完成）
3. **右键菜单**：待办暂不支持右键菜单
4. **提醒功能**：待办暂不支持到期提醒

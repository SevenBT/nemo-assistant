## 快速验证

```bash
python test_preset_edit.py
```

**预期结果**：所有测试显示 `[OK]`

## 功能概览

### 可以做什么
- ✅ 直接编辑内置预设（名称、图标、提示词）
- ✅ 保存修改
- ✅ 恢复到默认状态
- ✅ 创建自定义预设
- ✅ 编辑自定义预设
- ✅ 删除自定义预设

### 不能做什么
- ❌ 删除内置预设（有保护）

## 视觉标记

- `⚠️ 🤖 默认助手` - 被修改的内置预设
- `🤖 默认助手` - 未修改的内置预设
- `🎨 我的助手` - 自定义预设

## 按钮说明

- **新建** - 创建新的自定义预设
- **删除** - 删除自定义预设（内置预设时禁用）
- **导入** - 从 JSON 文件导入预设
- **导出** - 导出自定义预设到 JSON 文件
- **保存** - 保存当前编辑
- **恢复默认** - 恢复内置预设到原始状态（仅被修改时显示）

## 常见问题

### Q: 为什么内置预设不能删除？
A: 为了保持系统稳定性，内置预设不可删除，但可以编辑。如果不需要某个内置预设，可以忽略它。

### Q: 如何撤销对内置预设的修改？
A: 点击"恢复默认"按钮即可恢复到原始状态。

### Q: 修改内置预设会影响其他用户吗？
A: 不会。修改只保存在本地配置文件中。

### Q: 如何备份我的预设？
A: 使用"导出"功能将自定义预设导出到 JSON 文件。

## 技术细节

### 数据存储
- 配置文件：`~/.config/assistant/presets.json`
- 原始内置预设：内存中保存（不写入文件）

### 修改检测
比较以下字段：
- 名称（name）
- 图标（icon）
- 提示词（system_prompt）
- 参数（params）

### 恢复机制
从内存中的原始副本恢复，确保数据完整性。

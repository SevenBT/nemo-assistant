## 代码改动总结

### 1. PresetManager (app/core/preset_manager.py)
- 添加 `_builtin_originals` 字典保存原始内置预设
- 添加 `_save_builtin_originals()` 方法
- 修改 `update()` 方法：移除 `is_builtin` 检查
- 修改 `delete()` 方法：保留删除保护，但提示可编辑
- 添加 `restore_builtin()` 方法：恢复内置预设到原始状态
- 添加 `is_modified()` 方法：检查内置预设是否被修改

### 2. PresetManagerDialog (app/ui/preset_manager_dialog.py)
- 移除"复制"按钮
- 移除只读提示标签
- 添加"恢复默认"按钮（黄色背景，仅内置预设且被修改时显示）
- 修改 `_load_list()`：被修改的内置预设显示 ⚠️ 图标
- 修改 `_on_selection_changed()`：所有预设都可编辑，内置预设不可删除
- 添加 `_on_restore()` 方法：恢复内置预设
- 修改 `_on_save()` 方法：保存后重新选中以更新 UI 状态
- 移除 `_on_duplicate()` 方法

## 设计原则

1. **用户友好**：允许直接编辑内置预设，无需复制
2. **安全网**：提供"恢复默认"功能，防止误操作
3. **视觉反馈**：⚠️ 图标标记被修改的内置预设
4. **删除保护**：内置预设仍然不可删除，保持系统稳定性
5. **状态同步**：保存后自动更新 UI 状态（图标、按钮可见性）

# 截图 AI 流程

> [English](../en/screenshot-ai.md)

Nemo Assistant 有两条相互独立的截图路径。

## 本地 OCR

当用户想从图片中提取文字时使用本地 OCR。它通过 RapidOCR / ONNX Runtime 运行，不需要 LLM 服务商。

典型用法：

1. 按 `Ctrl+Alt+A`。
2. 框选屏幕区域。
3. 在截图操作中选择 OCR。
4. 使用或复制识别出的文字。

## 视觉模型分析

当用户希望所选模型直接理解图片本身时使用视觉分析。这需要：

- 一个已配置的 LiteLLM 模型。
- 保存在系统 keyring 中的服务商 API Key。
- 所选模型已启用或被自动检测为具备视觉能力。

典型用法：

1. 按 `Ctrl+Alt+A`。
2. 框选屏幕区域。
3. 选择 AI／视觉操作。
4. Nemo Assistant 会把图片附件发送到当前对话轮次。

## 为什么要分开

OCR 与视觉解决的是不同问题。OCR 在本地提取文字，更便宜、更快，且默认保护隐私。视觉分析更适合处理版式、图表、UI 截图，以及仅靠提取文字不够的图片。
